engage.us

to be addicted to good cause

Web App

The web application provides a platform that makes social issues more personal.

It revolutionizes the interaction between the community and organizations through crowdsourcing.

It encourages engagement in civic activities using virtual rewards, and it transforms volunteering activities to social games that participants can engage their friends with.

Mobile App

The mobile application is complementary to the web app. Users can use the mobile app to see itineraries of activities, and record events on social network sites.
